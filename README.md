# ScamCheck - Công cụ hỗ trợ nhận diện tin nhắn lừa đảo cho người cao tuổi

## Giới thiệu

ScamCheck giúp người dùng trên 45 tuổi kiểm tra nhanh các tin nhắn nghi ngờ trong vòng 30 giây.

## Công nghệ sử dụng

- HTML
- Tailwind CSS
- JavaScript thuần
- Google Gemini API (`gemini-3.5-flash`)

## Hướng dẫn cài đặt

1. Sao chép `config.example.js` thành `config.js`.
2. Điền Google Gemini API Key của bạn vào hằng số `GEMINI_API_KEY` trong `config.js`.

## Cảnh báo pháp lý

ScamCheck là công cụ giáo dục do nhóm học viên phát triển và đánh giá của ứng dụng không thay thế cảnh báo chính thức từ ngân hàng hoặc cơ quan chức năng.
